La Classe Buggle

Bienvenue dans la première leçon de programmation orientée objet !

Le but de cette leçon est de recréer le monde des buggles que vous avez déjà appréhendé lors de vos premierès leçons de java.

Pour ce premier exercice, vous allez devoir recréer la classe Buggle. Cette classe doit conserver deux entiers x et y qui représentent les coordonnées du buggle sur la grille.

Vous écrirez un constructeur Buggle(int px, int py) qui permet de créer un nouveau buggle et initialise les variables x et y aux valeurs respectives px et py.

Vous écrirez également les méthodes getX() et getY() qui retournent la valeur des paramètres x et y.

Enfin vous écrirez les méthodes setX(int x) et setY(int y) qui permettent de modifier la valeur des entiers x et y.

Si votre code est correct, vous verrez apparaitre un buggle dans la grille.
