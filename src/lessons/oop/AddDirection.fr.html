Direction

Vous vous souvenez surement que les buggles pouvaient se tourner dans différentes directions. Nous allons donc ajouter la possibilité de faire pivoter notre buggle

Pour cela, vous allez devoir utiliser un paramètre direction. Nous allons coder les différentes directions à l'aide d'un entier direction qui vaudra :

  • 0 pour le nord
  • 1 pour l'est
  • 2 pour le sud
  • 3 pour l'ouest

Vous modifirez votre constructeur Buggle(int px, int py) afin que le buggle soit par défaut tourné vers le nord. Puis vous ajouterez un deuxième constructeur MyBuggle(int px, int py, int pdirection) qui permet de créer un buggle en choisisant sa direction initiale.

Vous créerez également le setter void setDirection(int direction) et le getter int getDirection() pour ce nouveau paramètre .

Pour le moment, vous n'avez pas à gérer les cas où la direction n'est pas comprise entre 0 et 4. Nous verrons par la suite comment gérer ce genre de situation.

Si votre code est correct, le buggle sera un tour sur lui même dans sa case .
